cURL Commands for Todo API

Operation cURL Command
Get All Todo Items curl -X GET http://localhost:8000/api/todo/ -H "Authorization: Token {{ token }}"
Create a Todo Item curl -X POST http://localhost:8000/api/todo/ -H "Authorization: Token {{ token }}" -d "taskName=New Task" -d "taskDescription=<Description of the task>"
Delete a Todo Item curl -X DELETE http://localhost:8000/api/todo/<task id>/ -H "Authorization: Token {{ token }}"
Update a Todo Item Description curl -X PUT http://localhost:8000/api/todo/<task id>/ -H "Authorization: Token {{ token }}" -d "taskDescription=<Updated description>"
Mark Deleted API curl -X PATCH http://localhost:8000/api/todo/<task id>/mark_completed/ -H "Authorization: Token {{ token }}"
